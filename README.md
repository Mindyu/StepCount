已实现功能：
- 计步功能
- 锻炼计划及提醒功能
- 用户注册登陆
- 用户信息显示
- 步数历史信息

使用技术

- SQLite数据库
- Litepal框架
- SharePerference
- 安卓网络编程
- 异步请求 AsyncTask
- 头像图片选择
- Material Design 设计规范



后端系统

- Spring Boot微服务架构 整合 Spring Data JPA
- MySQL 数据库
